# Agro-AI Kurs: Colab-Notebooks

Dieses Repository enthält drei kleine Unterrichtsblöcke für Google Colab.

**Anleitung Colab:** Link öffnen, dann in Colab `Runtime > Run all` ausführen.
Die kleinen Datensätze werden automatisch geladen.

**Alternative ohne Google-Konto:** Binder-Link öffnen und warten, bis die Jupyter-Umgebung startet.
Beim ersten Start kann Binder einige Minuten brauchen.

## Technischer Hinweis zu Binder

Binder ist kostenlos und braucht kein Google-Konto, aber es ist nicht so schnell und stabil wie Colab.
Für die ersten zwei Blöcke ist Binder sehr gut geeignet. Der dritte Block mit PyTorch kann beim ersten Start länger dauern.
